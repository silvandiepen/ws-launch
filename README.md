# Web Socket Mock Server

### Installation

To start the project,

```
npm install
```

### Run

```
npm start
```

### Settings

| Settings | Default         | Argument |
| -------- | --------------- | -------- |
| Port     | `4000`          | `--port` |
| Data     | `./mockdata.js` | `--data` |
| Wait     | `5000`          | `--wait` |

If you want any other data, you can pass arguments like;

```
npm start -- --port=5000 --data=./test.json
```
